**Preliminary: Free action → H*\_{Z/4}(X̄, ∂X̄; —) \= H*(B, ∂B; —) per G17.** Since the Z/4 action on X̄ \= S¹ × Δ³ is free (established in Task A and v11a), equivariant cohomology of the pair equals ordinary cohomology of the quotient pair: H*\_{Z/4}(X̄, ∂X̄; M) \= H*(B, ∂B; M) for any coefficient module M.

**B1 — H⁴(B, ∂B; ℤ) (untwisted integer coefficients):**

B is a compact connected nonorientable 4-manifold with boundary ∂B. By Poincaré-Lefschetz duality for nonorientable manifolds: H⁴(B, ∂B; ℤ) ≅ H₀(B; 𝒪\_B) where 𝒪\_B is the orientation local system of B. Since B is nonorientable, 𝒪\_B is nontrivial (the sign local system). H₀(B; 𝒪\_B) is the 𝒪\_B-twisted zeroth homology of B. For a connected nonorientable manifold, H₀(B; 𝒪\_B) \= 0 (the only 0-cycles are points, and the orientation local system is nontrivial, so the twisted homology group is ℤ/2 in the twisted case — but the duality is with 𝒪\_B-twisted homology, and for the untwisted cohomology):

More directly: For a compact connected n-manifold with boundary (orientable or not), H^n(M, ∂M; ℤ) ≅ ℤ if M is orientable, and H^n(M, ∂M; ℤ) \= 0 if M is nonorientable.

This is a standard result: the fundamental class \[M, ∂M\] exists in H\_n(M, ∂M; ℤ) only when M is orientable; when nonorientable, it exists only in H\_n(M, ∂M; ℤ/2). Dually, H^n(M, ∂M; ℤ) \= 0 when M is nonorientable.

**Result: H⁴(B, ∂B; ℤ) \= 0\.** (Consistent with FLAG-3a, FLAG-3b conclusions — confirmed.)

**B2 — H⁴(B, ∂B; 𝒪\_B) (twisted by base orientation local system):**

By Poincaré-Lefschetz duality with twisted coefficients: H⁴(B, ∂B; 𝒪\_B) ≅ H₀(B; ℤ) ≅ ℤ (since B is connected and the double twist 𝒪\_B ⊗ 𝒪\_B ≅ ℤ returns to the trivial system).

**Result: H⁴(B, ∂B; 𝒪\_B) ≅ ℤ.** This group is nonzero. If the relevant Euler class lives in this group, the twisted-coefficient route is potentially viable.

**B3 — H⁴(B, ∂B; ℤ/2) (mod-2 coefficients):**

ℤ/2 coefficients are self-dual: every manifold is "orientable" over ℤ/2. By Poincaré-Lefschetz duality over ℤ/2: H⁴(B, ∂B; ℤ/2) ≅ H₀(B; ℤ/2) ≅ ℤ/2 (since B is connected).

**Result: H⁴(B, ∂B; ℤ/2) ≅ ℤ/2.** This group is nonzero. The mod-2 route has a nonzero target group.
