**Preliminary: Free action → H*\_{Z/4}(X̄, ∂X̄; —) \= H*(B, ∂B; —) per G17.** Since the Z/4 action on X̄ \= S¹ × Δ³ is free (established in Task A and v11a), equivariant cohomology of the pair equals ordinary cohomology of the quotient pair: H*\_{Z/4}(X̄, ∂X̄; M) \= H*(B, ∂B; M) for any coefficient module M.

**B1 — H⁴(B, ∂B; ℤ) (untwisted integer coefficients):**

B is a compact connected nonorientable 4-manifold with boundary ∂B. By Poincaré-Lefschetz duality for nonorientable manifolds: H⁴(B, ∂B; ℤ) ≅ H₀(B; 𝒪\_B) where 𝒪\_B is the orientation local system of B. Since B is nonorientable, 𝒪\_B is nontrivial (the sign local system). H₀(B; 𝒪\_B) is the 𝒪\_B-twisted zeroth homology of B. For a connected nonorientable manifold, H₀(B; 𝒪\_B) \= 0 (the only 0-cycles are points, and the orientation local system is nontrivial, so the twisted homology group is ℤ/2 in the twisted case — but the duality is with 𝒪\_B-twisted homology, and for the untwisted cohomology):

More directly: For a compact connected n-manifold with boundary (orientable or not), H^n(M, ∂M; ℤ) ≅ ℤ if M is orientable, and H^n(M, ∂M; ℤ) \= 0 if M is nonorientable.

This is a standard result: the fundamental class \[M, ∂M\] exists in H\_n(M, ∂M; ℤ) only when M is orientable; when nonorientable, it exists only in H\_n(M, ∂M; ℤ/2). Dually, H^n(M, ∂M; ℤ) \= 0 when M is nonorientable.

**Result: H⁴(B, ∂B; ℤ) \= 0\.** (Consistent with FLAG-3a, FLAG-3b conclusions — confirmed.)

**B2 — H⁴(B, ∂B; 𝒪\_B) (twisted by base orientation local system):**

By Poincaré-Lefschetz duality with twisted coefficients: H⁴(B, ∂B; 𝒪\_B) ≅ H₀(B; ℤ) ≅ ℤ (since B is connected and the double twist 𝒪\_B ⊗ 𝒪\_B ≅ ℤ returns to the trivial system).

**Result: H⁴(B, ∂B; 𝒪\_B) ≅ ℤ.** This group is nonzero. If the relevant Euler class lives in this group, the twisted-coefficient route is potentially viable.

**B3 — H⁴(B, ∂B; ℤ/2) (mod-2 coefficients):**

ℤ/2 coefficients are self-dual: every manifold is "orientable" over ℤ/2. By Poincaré-Lefschetz duality over ℤ/2: H⁴(B, ∂B; ℤ/2) ≅ H₀(B; ℤ/2) ≅ ℤ/2 (since B is connected).

**Result: H⁴(B, ∂B; ℤ/2) ≅ ℤ/2.** This group is nonzero. The mod-2 route has a nonzero target group.

## **PATH 1 — TWISTED-COEFFICIENT RELATIVE COHOMOLOGY**

**Objective:** Determine whether a twisted-coefficient or local-coefficient system can recover a nonzero relative class capable of forcing a zero of Φ\_min.

### **Step 1.1 — Fiber Orientation Character Audit (H31, MR4)**

Per H31, the relevant coefficient system for an Euler-class forcing argument is determined by the orientation local system 𝒪\_E of the vector bundle fiber — not by the base orientation sheaf 𝒪\_B — unless these are proved equal.

The fiber representation is ρ\_total \= χ₂^{ℝ²} ⊕ χ₃. We must compute det(ρ\_total(σ)) for the generator σ.

**χ₂^{ℝ²} action on ℝ²:** The generator σ acts by scalar multiplication by −1 on ℝ². As a 2×2 real matrix, this is −I₂. Its determinant is det(−I₂) \= (−1)² \= \+1.

**χ₃ action on ℂ ≅ ℝ²:** The generator σ acts by multiplication by −i \= e^{3πi/2}. As a 2×2 real matrix (encoding complex multiplication), this is the rotation matrix R(3π/2) \= R(−π/2). Its determinant is \+1 (rotations are orientation-preserving).

**Total determinant:** det(ρ\_total(σ)) \= det(−I₂) · det(R(−π/2)) \= (+1)(+1) \= \+1.

The fiber representation is **orientation-preserving** on the generator, hence on all group elements (since Z/4 is generated by σ). Therefore 𝒪\_E is **trivial** (constant ℤ sheaf).

**H31 consequence:** The Euler class e(ρ\_total) lives in H⁴(X̄/Z₄, ∂X̄/Z₄; ℤ) with **untwisted** ℤ-coefficients — not in any twisted group. The twisted-coefficient escape route requires 𝒪\_E ≠ 𝒪\_B; this requires the fiber representation to be orientation-reversing. Since the fiber representation is orientation-preserving, this route is closed.

### **Step 1.2 — Assessment of Base Orientation Twisted Route**

Could one instead use the base orientation sheaf 𝒪\_B to define a twisted relative cohomology group H⁴(X̄/Z₄, ∂X̄/Z₄; 𝒪\_B)?

By Poincaré-Lefschetz duality for a compact 4-manifold-with-boundary: H⁴(M, ∂M; 𝒪\_B) ≅ H₀(M; ℤ) ≅ ℤ (for connected M, regardless of orientability of M). So this group is nonzero.

However, per H31: the Euler class e(E) lives in H⁴(B, ∂B; 𝒪\_E), not H⁴(B, ∂B; 𝒪\_B), unless 𝒪\_E \= 𝒪\_B. Since 𝒪\_E is trivial and 𝒪\_B is nontrivial (the quotient is nonorientable), these sheaves are not equal. The class 2u² is an element of H⁴(X̄/Z₄, ∂X̄/Z₄; ℤ) \= 0\. It cannot be reinterpreted as a class in H⁴(X̄/Z₄, ∂X̄/Z₄; 𝒪\_B) ≅ ℤ without a separate argument identifying a nonzero element of that group with a geometric forcing obstruction. No such argument is available in this traversal.

**H31 certificate (Path 1):** The twisted-base-orientation route does not supply a forcing class for Φ\_min. The Poincaré-Lefschetz duality class in H⁴(M, ∂M; 𝒪\_B) ≅ ℤ is a fundamental-class object, not an Euler class of the vector bundle E associated to ρ\_total. These are distinct cohomological objects.

### **Step 1.3 — Survey of Local-Coefficient Alternatives**

Could a different coefficient system — for instance, H\*(X̄/Z₄, ∂X̄/Z₄; ℤ\[Z/4\]) or rational coefficients — recover a nonzero relative class?

**Rational coefficients:** H⁴(X̄/Z₄, ∂X̄/Z₄; ℚ). For a nonorientable compact 4-manifold-with-boundary, the top relative cohomology with ℚ-coefficients also vanishes (since ℚ is a field of characteristic 0 and the Poincaré-Lefschetz isomorphism requires orientation for untwisted top-degree cohomology). Specifically, H⁴(X̄/Z₄, ∂X̄/Z₄; ℚ) \= 0 for the same reason as ℤ: the generator of Z/4 acts orientation-reversingly.

**ℤ/2-coefficients:** This is Path 2 (handled separately below).

**ℤ\[Z/4\]-coefficients:** This would be the equivariant cohomology of (X̄, ∂X̄) before passing to the quotient — i.e., H*\_{Z/4}(X̄, ∂X̄; ℤ). Since the action is free, this equals H*(X̄/Z₄, ∂X̄/Z₄; ℤ) by the free-action principle. This is the same group already computed to be 0 in degree 4\.

### **Path 1 Conclusion**

**STATUS: CONCLUDED.** No twisted-coefficient or local-coefficient system produces a nonzero relative forcing class for the Euler class of Φ\_min. The underlying obstruction is the orientation-preserving fiber representation combined with the nonorientable quotient pair. Specifically:

* The fiber representation is orientation-preserving ⟹ 𝒪\_E is trivial ⟹ the Euler class lives in untwisted H⁴ \= 0\.

What would be needed is either: (a) a different compactification in which the orientation-reversing stratum is collapsed or removed, or (b) a different forcing theorem not based on the Euler class (e.g., Fadell-Husseini index, relative Lusternik-Schnirelmann category, or a non-characteristic-class degree argument).

* H⁰(BZ/4; ℤ/2) \= ℤ/2  
* H¹(BZ/4; ℤ/2) \= ℤ/2, generated by a class a (the mod-2 reduction of the fundamental class)

### **Step 2.2 — Stiefel-Whitney Classes of ρ\_total**

The Stiefel-Whitney classes live in H\*(BZ/4; ℤ/2). The total bundle has fiber representation ρ\_total \= χ₂^{ℝ²} ⊕ χ₃^{ℝ²} as a real rank-4 bundle (viewing the ℂ-line χ₃ as a real ℝ²-plane).

**w₂ computation:** For a complex line bundle L with c₁(L) \= k·u in H²(BZ/4; ℤ), its mod-2 reduction is the second Stiefel-Whitney class w₂(L) \= c₁(L) mod 2\.

* For χ₂^{ℝ²} viewed as the real plane bundle with σ acting by −I₂: as a complex line, c₁ \= 2u, so w₂(χ₂^{ℝ²}) \= 2u mod 2 \= 0\.  
* For χ₃ (complex line bundle): c₁(χ₃) \= 3u \= −u (mod 4), so w₂(χ₃) \= 3u mod 2 \= u mod 2 \= a² ... but wait: per the ring structure, H²(BZ/4; ℤ/2) is generated by b, and the mod-2 reduction map H²(BZ/4; ℤ) → H²(BZ/4; ℤ/2) sends u to b. So w₂(χ₃) \= 3u mod 2 \= u mod 2 \= b ∈ H²(BZ/4; ℤ/2).

By the Whitney product formula: w₂(ρ\_total) \= w₂(χ₂^{ℝ²}) \+ w₂(χ₃) \= 0 \+ b \= b ≠ 0\.

**w₄ computation:** The top Stiefel-Whitney class w₄ is the mod-2 Euler class.

For a direct sum E \= E₁ ⊕ E₂ with ranks 2 and 2: w₄(E) \= w₂(E₁)·w₂(E₂) \+ w₁(E₁)·w₃(E₂) \+ ... (Whitney product formula, full expansion). Since w₁ \= 0 for both (both orientation-preserving):

w₄(ρ\_total) \= w₂(χ₂^{ℝ²})·w₂(χ₃) \+ higher terms involving w₃.

Since w₂(χ₂^{ℝ²}) \= 0:

w₄(ρ\_total) \= 0·b \+ (w₁(χ₂^{ℝ²})·w₃(χ₃) \+ w₀·w₄(χ₃)) \= 0 \+ 0 \+ w₄(χ₃).

But χ₃ is a complex line bundle, hence a real rank-2 bundle. A rank-2 real bundle has w₃ \= 0 and w₄ \= 0 (Stiefel-Whitney classes vanish above the rank). Therefore w₄(χ₃) \= 0\.

**w₄(ρ\_total) \= 0\.**

### **Step 2.3 — Relative Mod-2 Class**

Actually, more precisely: for a rectangle with diagonals d₁ and d₂, perpendicularity means d₁ ⊥ d₂, i.e., Re(d₁·\\overline{d₂}) \= 0\.
